Standard Commands

Q: What are standard commands?

A: This category includes commands which can generally be used in any scenario and will only affect you.

Use chat commands like so: /<command> <argument(s)>. Some commands may not have arguments.
Example no-argument command: /quit
Example command with arguments: /join myroom

Other command types:

  • Operator commands (for running rooms)
  • Option commands
  • Set commands


Q: Where can I see all the commands?

A: While ingame, you can bring up a full list of all commands using the /fh command. Here's a more descriptive list of those commands:

Some of these commands may not work in certain versions of Toribash

Common commands
  • COMMANDARGSSHORTHANDDESC
    join[roomname]joJoins you to the room with the specified name.
    If no room of this name exists, it will be created for you and you will be opped.
    reconnect-recReconnects your client to the last room you were in.
    If you are already in a room, you will first be disconnected.
    spec-spMoves you from the bout queue (people in queue to play) to spectators.
    enter-enMoves you from the spectators into the bout queue (people in queue to play).
    bet[TC][username]Places a bet on the specified user (user must be currently playing).
    The amount betted must be above the room's minbet value, and the bet must be placed within the betframes.
    autobet[tori|uke|off][TC]Causes your client to automatically bet the specified TC amount on either Tori or Uke at the start of each match.
    chatlines[number]chlSets the number of displayed chat lines to the specified number (max 18).
    You can also use the ` (backtick) key to set this immediately to 18.
    pass[password]pwAllows you to play and chat in a password protected room.
    emote[message]emWhen playing, will display the specified message above your tori's head.
    shout[message]shCauses the specified message to appear at your camera's location.
    whisper[username][message]whOpens a new chat window that only you and the specified user will be able to see.
    The message argument is optional.
    hush-Disables whispers.
    login[username][password]loLogs you into the account specified.
    Must be spectating in multiplayer for this command to work.
    rememberme-rmCauses the client to remember your details and keep you logged in even if you close and reopen it.
    forgetme-fmCauses the client to forget your details after you close it.
    search[string]saSearches names of all online players to find matches with the specified string.

    For example, 'fe' would list all players who's tag+name contain the letters 'fe' in that combination.
    Use refresh before searching for more accurate results.
    refresh-rfSyncs local cache of rooms and online players with those actually online.
    stats[username]Displays the specified user's stats:
    • Join Date.
    • Belt.
    • Games played + win ratio.
    • Current wins/losses in room.
    • Global rank.
    quit-quExits the client.

Additional commands
  • COMMANDARGSSHORTHANDDESC
    rulesDisplay the rules of the room:
    • Tourney prize (for autotourneys).
    • Minimum bet.
    • Maximum bet rake.
    • Entry fee.
    • AFK penalty (TC lost for AFK'ing).
    • Decap prize.
    • Dismemberment prize.
    • Whether matches affect your rank.
    historyhiDisplays your entire chat log, including commands (inc. past sessions).
    refreshtorishoprtsUpdates local cache of torishop to live version.
    connect[address:port]coJoins you to the room with the specified addess:port. Must be an existing room. Useful for reconnecting to ghosted servers.
    listliLists all current rooms.
    Works best if used after /rf.
    download[username]dlDownloads the specified user's custom data to your computer, allowing you to view their current items properly.
    loadplayer[username][0|1]lpLoads the specified player's custom data onto either Tori or Uke:
    1. Tori.
    2. Uke.
    loadgui[username]lgLoads the specified player's GUI custom data to your client.
    addbuddy[username]abAdds the specified user to your buddies list.
    removebuddy[username]rbRemoves the specified user from your buddies list.
    listbuddylbLists all your current buddies.
    closetabctCloses the current chat tab.
    addtab[name]atNot working correctly.
    Opens a new chat tab.
    option[key [arg(s)]]optSets/gets the specified key's values.
    See the full list and details.
    zoomplayer[0|1]zpZooms the camera to the specified player:
    1. Tori.
    2. Uke.
    smoothcamera[0|1]smcNo effect.
    smartcamera[0|1]scNo effect.
    exportworld[filename].tbmewExports the current mod's world objects into a new mod file (.tbm file).
    exportrules[filename].tbmewExports the current mod's rules into a new mod file (.tbm file).
    resolution[width][height]reSets the game window's width and height (in pixels).
    anaglyphic[0|1]anNo effect.
    Would enable anaglyphic mode in versions such as 3.69.
    depthoffield[0|1]dofNo effect.
    Would presumably enable depth of field.
    worldshader[id][x][y][z][w]wsChanges the values of the field specified by [id] to the specified values.
    See shaders.
    loadworldshader[filename].inclwsImports the specified shader into a the client.
    exportworldshader[filename].incewsExports the current shader into a new shader file (.inc file).
    loadreplay[filename].rpllrLoads the specified replay file (.rpl file).
    If you are in multiplayer, you will be disconnected.
    savereplay[filename].rplsrSaves the current replay as a new replay file (.rpl file).
    loadflame[0|1][id1][id2][id3][id4]lfLoads the flames with the specified ids, and places them on Tori or Uke:
    1. Tori.
    2. Uke.
    cacheflamescfDownloads all existing flames to local flame cache.
    altimeter[0|1]amDisplays the altimeter (current altitude + record altitude).
    damagemeter[0|1]dmCrashes the game.
    Displays the damagemeter (current damage + record damage).
    speedometer[0|1]smDisplays the speedometer (current speed + record speed).
    scoringmeter[0|1]scmNo effect.
    benchmark[0|1]bmEnables benchmarking.
    invokecrashNo effect.
    clearbloodcbClears all blood.
    lobby[help|scan|use|list]lob
    • Help: shows other commands.
    • Scan: crashes the game.
    • Use: unknown effect.
    • List: lists current lobbies.
    gameserver[help|scan|use|list]gs
    • Help: shows other commands.
    • Scan: unknown effect.
    • Use: unknown effect.
    • List: lists current gameservers.
    steam[help|print|purchase [itemid]]st
    • Help: shows other commands.
    • Print: displays your steamId and username.
    • Purchase: Allows you to purchase the specified item through Steam.
      Item must have a USD pricetag.
    loadscript[filename].lualsLoads the specified Lua script (.lua file).
    loadmod[filename].tbmlsLoads the specified toribash mod file (.tbm file).
    removehooks[hookId]rhUnknown effect.
