Name changes and other TC sinks

Q: What is a namechange?

A: If you do not like the name you currently have in Toribash you can pay to get it changed.

Note: Your new name should be at least 4 characters long and consist only of alphanumeric characters.

Name Changes ►
  • Name changes can only be done by administrators. It costs 100,000 TC. This price doubles every time you change your name. If you only want to change the case of a letter in your username, it costs 5k per letter changed. For example, 'Fish' to 'FISH' costs 15k. If the name you want is already taken, that account must be both unused and inactive for at least 3 years. To check if a name is taken, type the name at the end of this link: http://forum.toribash.com/member.php?username= When you have found a suitable name, you may contact and administrator
Profile Customization ►
  • This is generally restricted to people who have purchased VIP. You can pay to get usertitles or avvy which you wouldn't usually be able to set with VIP. Usertitles, avvys and sigs are not permanent. The normal restrictions will apply whenever you try to change them yourself.
QI Transfers►
  • QI can not be traded between accounts for any price. No exceptions.
Buying QI►
  • QI can be purchased here, or you can purchase a QI booster that will award you with 2 QI instead of 1 for each game played. Buy QI boosters here.
